<h2>战舰游戏简介</h2>
游戏规则及描述：在游戏中玩家有5次机会猜出敌舰的坐标位置（横纵坐标都在0-7），战舰有身体、头部尾部之分，如果猜中身体坐标，则认为击中，直接胜利，如果击中头部或者尾部，则战舰的血量减一，知道机会用完，或者战舰被击沉，游戏结束，游戏运行截图如下
